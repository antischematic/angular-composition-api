# Angular Phalanx

Atomic state management for Angular Composition API. [Learn More](https://mmuscat.github.io/angular-composition-api/docs/store/intro)

---

## Quick Start

Install via NPM

```bash
npm install @mmuscat/angular-phalanx
```

Install via Yarn

```bash
yarn add @mmuscat/angular-phalanx
```
